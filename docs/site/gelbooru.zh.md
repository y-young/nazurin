# Gelbooru

## 自定义存储路径和文件名

更多信息请查阅 [自定义存储路径和文件名](./index.zh.md/#customizing-storage-path--file-name)。

### GELBOORU_FILE_PATH

:material-lightbulb-on: 可选，默认为 `Gelbooru`

存储路径。

### GELBOORU_FILE_NAME

:material-lightbulb-on: 可选，默认为 `{id}`

文件名称。
