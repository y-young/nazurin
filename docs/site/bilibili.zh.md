# Bilibili 动态

<https://t.bilibili.com/>

## 自定义存储路径和文件名

更多信息请查阅 [自定义存储路径和文件名](./index.zh.md/#customizing-storage-path--file-name)。

### BILIBILI_FILE_PATH

:material-lightbulb-on: 可选，默认为 `Bilibili`

存储路径。

### BILIBILI_FILE_NAME

:material-lightbulb-on: 可选，默认为 `{dynamic_id_str}_{index} - {user[name]}({user[uid]})`

文件名称。

```json
{
  "item": {

  },
  "user": {
    "name": "用户名称",
    "uid": "用户ID"
  },

}
```
